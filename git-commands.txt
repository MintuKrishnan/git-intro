Git Commands :
1. To Check the version : 
   git --version
2. Initialized empty Folder :
   git init
3. Check the Staus ::
   git status
4. Add file into stage :
   git add filename
5. Add all files into stage :
   git add .
6. Commit the work :
   git commit -m "Done"
7. Check log entry :
   git log
8. Add user name and email :  
   git config --global user.name "shubham"
   git config --global user.email "gshubham934@gmail.com
9. Edit by mac command line :
   nano filename
